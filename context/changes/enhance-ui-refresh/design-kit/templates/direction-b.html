Darkroom — precision instrument

The UI as a digital darkroom tool: neutral charcoal with no blue cast so photographs read true, hairline borders instead of shadows, and monospaced readouts that treat every parameter like an exposure setting. Cyan is the working color; violet exists only on the primary action. Key-visual treatment: palette-only — no imagery behind the interface, the user's photo is the only picture on screen.
